What We Detect
Comprehensive PII Coverage
From government IDs to biometric hashes — our three-pass AI engine identifies sensitive data across every format.

Identity Documents

Aadhaar, PAN Card, Passport numbers — detected via specialized Indian document recognizers.

Contact Information

Phone numbers, email addresses, physical addresses — caught by regex and NER models.

Financial Data

Bank accounts, IFSC codes, UPI IDs, credit card numbers — with contextual validation.

Biometric Hashes

Fingerprint hashes, face templates, device IDs — pattern-matched with high precision.

Personal Details

Names, dates of birth, IP addresses — identified through spaCy NER and keyword context.

AI Deep Learning

DeBERTa-v3 NER model catches what regex misses — names, addresses, and context-dependent PII.

Security & Architecture
Built for Privacy
  • Fernet encryption at rest — all uploaded files are encrypted immediately using symmetric AES encryption.
  • JWT authentication with bcrypt password hashing — industry-standard auth.
  • Role-based access control — Admin and User roles with separate permissions.
  • Complete audit trail — every upload, download, login, and PII detection is logged.
  • Three-pass detection — Presidio regex + Keyword context + DeBERTa AI for maximum coverage.
  • Self-hosted — all processing happens server-side. No data leaves your infrastructure.
